## mindary - A virtual dictionary

# What is it for?
- mindary, is a virtual dictionary. You can write things that you can forget. When you searched from mindary, it helps to you remember.

# How to use?
- Firstly, write your words to tests/dic.txt by line to line.
- Then, enter build program and it works.

# How to build?
- This program is craated by using Rust. So you need ```cargo``` package.
- Open up ```CMD``` in mindary's folder.
- Use ```cargo build --release```.
- Copy the ```tests/``` folder to ```mindary/target/build```.
- Open up ```mindary.exe``` and then it will work.
- (Or you can use ```cargo run``` to open it in ```CMD```.)

Thank you for using!
